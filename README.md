# Sialkot Sportswear Template Catalog

50 unique static website templates for sportswear manufacturing and export businesses in Sialkot, Pakistan.

## Quick Start

Open any `templates/XX-name/index.html` directly in a browser. Each template fetches `data/content.json` for content.

## Structure

```
data/content.json       ← Shared content (edit once, applies to all templates)
templates/
  01-champion-heritage/ ← Template folder
    index.html          ← Standalone page
  02-forge-performance/
    index.html
  ... (50 total)
```

- HTML5 + Tailwind CSS (CDN) + Vanilla JS
- No build tools required
- Responsive (mobile → desktop)
- Placeholder images via Unsplash
